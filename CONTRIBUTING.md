# Contributing

This repo teaches people to build with AI agents. Quality bar:

- **Every code block must run.** Test it. If it needs an API key or Ollama, say so.
- **Show tradeoffs.** Don't pitch a framework as a silver bullet.
- **Local-first when possible.** Prefer free/private options (Ollama) over paid clouds.
- **No hype.** Explain *why*, not just *how*.

## How to contribute
1. Fork and branch (`fix/02-agent` or `add/10-evals`).
2. Add or edit a tutorial folder. Keep the `NN-name/README.md` convention.
3. Update the table in `README.md`.
4. Open a PR with a one-line description of what changed and why.

Corrections to existing tutorials are just as valuable as new ones.
